# Bomberman JS remake

_This is a work in progress_

Single player JavaScript remake of Bomberman game for NES console (1983) using HTML5 canvas.

## Setup

Navigate to cloned repo's directory and run:

```
npm install
```

To run dev server

```
npm run start
```

To build app

```
npm run build
```
