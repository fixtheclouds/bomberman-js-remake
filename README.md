# Bomberman JS remake
[Work in progress]

Single player Javascript/HTML5 remake of Bomberman game for NES console (1983)


## Stack

- HTML5 Canvas
- Javascript (ES6)
- LoDash
- Webpack
- Mocha (testing)

## Setup

Navigate to cloned repo's directory and run:

`npm install`

To run Webpack

`npm run build`
